## Rappels des deux tokens manipulés ici


POST `/auth/login`
- > accessToken 1H
- > refreshToken 7J


`Access Token` = 
- permet d'avoir les accès à toutes les routes de l'APP
- ne peut pas être invalidé (d'où la durée d'1h)
- stockée dans les cookies
- transféré au backend à chaque requête `/*` via : 
	- soit les cookies
	- soit le header `Authorization: Bearer XXXXX` 


`Refresh Token` = 
- permet d'avoir un nouvel access token
- on l'envoie uniquement à la route `/refresh` via :
	- soit les cookies (il s'envoie uniquement sur cette route la automatique grâce à l'attribut `path`)
	- soit le body { refreshToken: XXXX }


## Quelle approche choisir pour O'quiz ?

Besoin : 
- Rôles : visiteurs, membres connectés, auteurs, administrateurs
- Règles : (ex) visiteurs peut voir les quiz les plus récents, membres connectés jouer des quiz, auteurs créer un quiz, administrateur créer des niveaux

- Rôle : peu, fixes
- Règle : fixes ==> permissions codés en dur 
- 1 utilisateur === 1 rôle
  - ==> attribut `rôle` sur l'utilisateur serait probablement le plus simple (ENUM : `member` | `author` | `admin`)

- où encoder les permissions ? 
  - au niveau des routes ? => ✅ car notre API est petite, très couplé à notre front, 
  - au niveau des controlleurs/lib ? => complexifierait notre approche, mais pour certains contrôles, on aura pas le choix (ex: les auteurs ne peuvent modifier QUE leur propre tags)

## Axios

Intérêt n°1 : syntaxe simplifié

```js
// GET (fetch)
const httpResponse = await fetch(URL);
const data = await httResponse.json(); // Notre body est dans "data"

// GET (axios)
const { data } = await axios.get(URL); // Notre body est dans "data"
```

```js
// POST (fetch)
const httpResponse = await fetch(URL, {
  method: "POST",
  headers: { "Content-Type": "application/json" },
  body: JSON.stringify(BODY)
});
const data = await httpResponse.json();

// POST (axios)
const { data } = await axios.post(URL, BODY);
```

Intérêt n°2 : 

Possibilité de créer des instances axios pré-configuré 

```js
const httpRequester = axios.create({
  baseURL: "http://localhost:3000/api",
  headers: { "Authorization": `Bearer ${TOKEN}` }
});

const { data: createdUser } = await httpRequester.post("/users", { ... }); // Cette requête est automatiquement authentifié puisqu'on utilise le httpRequester pré-configuré
```
