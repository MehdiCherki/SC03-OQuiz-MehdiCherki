# Challenge SC01E01 - Conception

Bonjour et encore une fois, bienvenue chez O'Clock ! On vous souhaite le meilleur pour votre formation ! 🎉  

Pour votre premier jour, on vous demande de commencer la partie conception d'un projet, à partir d'une prise de notes fictive suite à un rendez-vous client.

## Contexte

Vous êtes un autoentrepreneur (un "freelancer" comme on dit chez les GenZ). Vos talents de développements hors pairs ne sont pas passés inaperçus et par bouche-à-oreille, un client vous a contacté. Après plusieurs appels et grâce à vos skills de *Salesdog* aguerri, vous êtes invité au siège social du client.  

À la suite de ce rendez-vous, vous décrochez la prestation (congrats 🍾).

- Préparer des questions pour éclaircir les zones d'ombre avec le client. Critiquer certains choix, apporter votre expertise en matière de développement et aider à la conception.

Pour vous aider dans cette tâche, vous retrouvez dans votre sacoche les notes prises lors du premier rendez-vous client gribouillées sur un bout de papier la semaine dernière.

## Notes meeting client

```
Réunion : 19 septempbre
Titre : notes 1er RDV client
Client : The Marvelous Corporation

- CEO veut site vitrine pour exposer des projets digitaux (publics, charités, etc. pas de projets privés/confidentiels). 
- Objectif : permettre aux clients potentiels de visualiser le portfolio (services, réalisations) de l'entreprise.
- Objectif : faciliter la prise de contact (réseaux etc)
- Projets s'affichent comme une liste de "cartes".
- CTO veut afficher le nom des technos utilisés pour chaque projet avec des pastilles sur les projets.
- Informations de contact en bas de la page du site, mentions légales etc...
- Pas de connexion pour les utilisateurs (site public vitrine), mais espace administrateur possible pour ajouter de nouveaux projets.
- Click sur un projet -> page de projet, affiche informations spécifiques au projet. 
- Technologies : raw HTML/CSS/JS (pas de framework sauf frameworks CSS type bootstrap). 
- Prise de contact via formulaire type email/message. 
- Logo customisable en haut du site vitrine, avec le nom de l'entreprise.
- Nom de l'outil : O'vitrine
```
